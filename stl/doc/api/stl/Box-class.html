Box

Box<T> class

A mutable wrapper designed to hold and update values.

Unlike Var<T> or Ref<T>, Box<T> is intensely overloaded, mimicking the robust flexibility of idiomatic Dart. If Box<T> holds a numeric type (int or double), you can mathematically interact with the Box natively without unboxing it explicitly.
