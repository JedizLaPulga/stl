Array

Array<T> class

A conceptually strict fixed-size contiguous sequence.

Designed directly to emulate C++ std::array, this wrapper entirely rejects dynamic memory boundary resizing. Any attempt to expand, dynamically scale, or shrink the internal arrays mathematically throws deterministic structural bounds errors!
