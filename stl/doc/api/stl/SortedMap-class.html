SortedMap

SortedMap<K, V> class

A collection that stores elements as key-value pairs, maintaining them in strictly sorted order by key.

In the C++ STL, this matches exactly the behavior, complexity, and contract of std::map. It utilizes a balanced tree structure (Splay tree) under the hood. Iteration yields elements in strictly mathematical or custom-comparator order of keys.
